## 示例 1：用户提供完整信息

```
用户: 数据在 /data/cmems，变量是 chl 和 no3，输出到 /output，4 倍下采样用 area 方法，划分 0.7/0.15/0.15

Agent: 收到，我先分析一下您的数据...

[分析数据]

数据分析完成：
- 文件数量: 365 个
- chl 形状: (365, 680, 1440)
- no3 形状: (365, 680, 1440)
- 检测到掩码: mask
- 检测到静态变量: lon, lat

请确认：
1. 掩码变量使用 mask 吗？
2. 需要保存哪些静态变量？

用户: mask 用 mask，静态变量保存 lon lat

Agent: 好的，执行前请确认所有参数：

【处理参数确认】
- 数据目录: /data/cmems
- 研究变量: chl, no3
- 掩码: mask
- 静态变量: lon, lat
- 下采样: 4x, area 方法
- 划分: 70%/15%/15%

确认无误后我将开始处理。

用户: 确认执行

Agent: [开始执行...]
```

## 示例 3：使用粗网格模式

```
用户: 我有两套数据，HR 在 /data/hr，LR 在 /data/lr，都是 ROMS 模型输出，帮我转成超分训练格式

Agent: 好的，您使用的是粗网格模式（HR/LR 数据由模型分别生成）。我先分析一下您的 HR 数据...

[分析 HR 数据]

HR 数据分析完成：
- 文件数量: 365 个
- 数据形状: (365, 680, 1440)
- 动态变量: temp, salt, uo, vo
- 静态变量: lon_rho, lat_rho, h, mask_rho

请问：
1. 您想研究哪些变量？
2. 输出目录是哪里？

用户: 研究 temp 和 salt，输出到 /output/dataset

Agent: 好的。还需要确认：
1. 掩码使用 mask_rho 吗？
2. 需要保存哪些静态变量？
3. 数据集划分比例？

用户: mask_rho，静态变量保存 lon_rho lat_rho，划分 0.7/0.15/0.15

Agent: 好的，执行前请确认所有参数：

【处理参数确认】
- 超分方式: 粗网格模式（HR/LR 分别转换）
- HR 数据目录: /data/hr
- LR 数据目录: /data/lr
- 输出目录: /output/dataset
- 研究变量: temp, salt
- 掩码: mask_rho
- 静态变量: lon_rho, lat_rho
- 划分: 70%/15%/15%

确认执行？

用户: 确认

Agent: [开始执行完整流程...]
```

**粗网格模式调用参数**：
```json
{
  "nc_folder": "/data/hr",
  "output_base": "/output/dataset",
  "dyn_vars": ["temp", "salt"],
  "user_confirmed": true,
  "mask_vars": ["mask_rho"],
  "stat_vars": ["lon_rho", "lat_rho"],
  "train_ratio": 0.7,
  "valid_ratio": 0.15,
  "test_ratio": 0.15,
  "lr_nc_folder": "/data/lr"
}
```

## 常见用户问法处理

| 用户说 | Agent 应该做什么 |
|--------|-----------------|
| "帮我预处理海洋数据" | 追问数据目录、输出目录，然后分析数据 |
| "数据在 /data/ocean，输出到 /out" | 先分析数据，展示变量让用户选择 |
| "数据在 /data，变量是 chl no3，输出到 /out" | 分析数据，追问处理参数 |
| "...4倍下采样，area方法，0.7/0.15/0.15" | 记下参数，执行前 check 确认 |
| "看看 /data 有什么变量" | 调用 ocean_inspect_data，只展示不处理 |
